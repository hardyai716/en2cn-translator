# en2cn Translator

> 浏览器扩展 — 英文网页段落级翻译助手

---

## 功能

- **段落级翻译**：鼠标悬停在英文段落上，浮出「译」按钮
- **点击触发**：手动点击，不自动翻译，不干扰阅读
- **原文保留**：译文出现在段落**下方**，原文不动
- **缓存复用**：同一段内容不重复请求 API
- **SPA 兼容**：React / Vue 等动态页面也支持
- **翻译引擎**：Google Translate（免费 API）

---

## 安装（开发模式）

### Chrome / Edge

1. 打开 `chrome://extensions`（或 `edge://extensions`）
2. 开启右上角「**开发者模式**」
3. 点击「**加载已解压的扩展程序**」
4. 选择本项目根目录 `en2cn-translator/`
5. 启用扩展，打开任意英文网页即可使用

---

## 使用方式

| 操作 | 效果 |
|------|------|
| 鼠标悬停到段落上（~300ms） | 段落右上角出现「译」圆形按钮 |
| 点击「译」按钮 | 译文插入段落下方，原文保留 |
| 滚动页面 | 按钮自动跟随段落重定位 |
| 段落滚出视口 | 按钮自动隐藏 |

│   ├── content/

## 技术栈

| 层 | 技术 |
|----|------|
| 扩展框架 | Manifest V3 |
| 页面交互 | 原生 DOM 事件（mouseover / mouseout 委托） |
| 翻译 API | Google Translate `translate.googleapis.com`（免费，无需 Key） |
| 数据持久化 | `chrome.storage.local`（译文缓存） |
| 动态页面 | `MutationObserver` 保活 |

## 常见问题

**Q: 为什么悬停后没有出现按钮？**
段落可能过短（少于 12 个英文字符）、或不在支持的元素列表中（仅支持 `<p>` / `<li>` / `<blockquote>` / `<h1-6>` / `<td>` / `<th>`）。

**Q: 翻译失败了怎么办？**
检查网络是否能访问 `translate.googleapis.com`。在大陆地区可能需要代理。失败时段落下方会有红色错误提示，3 秒后消失。

**Q: 能关闭翻译吗？**
当前版本通过「不悬停 / 不点击」即可完全避开功能。未来版本可添加 popup 开关。
